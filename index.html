Navbar 1 - Popup Bar

This is a simple popup style navbar. It is a compact, on-demand navigation menu that appears when needed, helping save screen space and offering easy access to important options.

Navbar 2 - DropDown Bar

This is a simple dropdown style navbar. It is a menu that drops down when clicked on it and providing access to more options in a compact format.

Navbar 3 - Side Bar

This is a simple dropdown style navbar. It is a menu located on the side of a webpage and only appear when we clicked on it, offering easy access to different sections or features, streamlining navigation.
